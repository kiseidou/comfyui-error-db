## エラーの概要

PyTorchを更新した後、ComfyUIで以下のようなエラーが発生します。

```
Could not locate cublasLt64_12.dll
```

このエラーはPyTorchやCUDAの依存関係に問題がある際に表示されます。特に、PyTorchの更新後によく見られます。

## 原因

- PyTorchを新しいバージョン（例：cu130）にアップデートしたが、必要なDLLファイル（`cublasLt64_12.dll`など）が存在しない。
- CUDA Toolkitがインストールされていないか、PATHが適切に設定されていない。

## 解決策

### 1. PyTorchとCUDAのバージョンを一致させる

PyTorchの更新後は、それに合わせてCUDAのバージョンも更新する必要があります。`cublasLt64_12.dll`のようなDLLファイルがインストールされていない場合、PyTorchの依存関係に問題がある可能性があります。

- **PyTorchをダウンロード**：公式サイトから最新のPyTorch（CUDA対応版）をダウンロードします。例えば、cu130に対応するバージョンを選択。
  
  ```bash
  pip install torch==2.9.1+cu130 torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/cu130
  ```

- **CUDA Toolkitをインストール**：PyTorchの依存関係に必要なCUDA Toolkitを公式サイトからダウンロードし、インストールします。

### 2. DLLファイルの手動コピー

場合によっては、DLLファイルが存在する別の場所から手動でコピーすることができます。例えば：

- `cublasLt64_12.dll`は通常、PyTorchインストールディレクトリの`torch\lib`フォルダ内にあります。

  ```bash
  cp path/to/your/torch/lib/cublasLt64_12.dll your_project_directory/
  ```

### 3. 環境変数を設定

CUDA Toolkitがインストールされている場合、環境変数のPATHを適切に設定します。例えば：

```powershell
@set CUDA_PATH=D:\AI\ComfyUI_windows_portable\python_embeded\Lib\site-packages\triton\backends\nvidia
```

## まとめ

`Could not locate cublasLt64_12.dll`のエラーはPyTorchとCUDAのバージョン整合性に起因することが多いです。DLLファイルが存在しない場合は、必要なファイルを手動でコピーするか、環境設定を確認して修正します。これにより問題が解決されることが多いです。

以上のように、それぞれのステップを追って実施することでComfyUIの動作が正常に戻ることが期待されます。
