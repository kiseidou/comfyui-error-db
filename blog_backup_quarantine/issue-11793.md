## エラーの概要

このエラーは、`gfx1200` が設定された環境で ComfyUI を実行した際に発生します。特に PyTorch Attention ノードを使用する場合に、初期設定で `Enable PyTorch Attention` がオフになっていると表示されます。

## 原因

このエラーは、ComfyUI の特定のバージョン（例えば PR #9926）で追加された `gfx1200` シリーズに対応する設定が不足しているか、PyTorch Attention ノードのデフォルト設定とインストール環境との相違によるものです。

## 解決策

### 1. 設定ファイルを修正する
既存の ComfyUI 設定ファイル（通常は `settings.yaml`）を開き、`gfx1200` のセクションが存在し、かつ PyTorch Attention ノードの初期設定として `Enable PyTorch Attention` が有効にされていることを確認します。

具体的な手順:
```yaml
# settings.yaml の該当部分を修正する
gpu_type: gfx1200
pytorch_attention_default_enabled: true
```

### 2. Python 環境を更新する
PyTorch バージョンが最新のものに更新されていることを確認します。

```bash
pip install torch --upgrade
```

### 3. ComfyUI を再インストールまたは更新する
必要に応じて、ComfyUI 自体を再インストールまたは最新版にアップデートします。

```bash
git pull origin main
python setup.py install
```

## まとめ

`Enable PyTorch Attention by default on gfx1200` のエラーは、設定ファイルの修正と Python 環境の更新により解決できます。問題が継続する場合は、ComfyUI の再インストールを試みてください。
